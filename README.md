# RuffoSpa

**Статус:** архивный / не поддерживается

Это старый JavaScript-проект (последняя активность около 2012—2014). Репозиторий сохранён для истории, но не поддерживается и не должeн использоваться в продакшене.

Что в репозитории
- Папка `www/` — исходники веб‑приложения (Frontend). Точные точки входа: смотрите `www/index.html` или аналогичные файлы.

Почему я добавил этот README
- Чтобы быстро понять назначение репозитория и его статус без открытия кода.
- Чтобы пометить проект как архивный и дать инструкции по восстановлению в будущем.

Если вы хотите восстановить проект
1. Перейдите в Settings → Options → Archive this repository (веб‑интерфейс) — это сделает репозиторий в режиме "только для чтения".
2. Чтобы снова работать с репо — нажмите Unarchive в тех же настройках.

Как запустить локально (быстрые шаги)
- Если в папке `www/` есть `index.html`, просто откройте файл в браузере или используйте статический сервер:

  npx http-server www -p 8080

- Если проект требует сборки — проверьте наличие `package.json` или другой инструкции в корне/`www/`.

Лицензия
